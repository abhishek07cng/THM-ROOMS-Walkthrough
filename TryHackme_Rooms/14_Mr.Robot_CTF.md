# Key Learnings

* Always inspect `robots.txt`.
* WordPress login responses can leak valid usernames.
* Theme editors can provide code execution when plugin uploads are blocked.
* Password hashes left on systems often lead to lateral movement.
* Legacy SUID binaries are common privilege escalation vectors.
* Enumeration is often more important than exploitation.
